Projects

Below are some of the projects that I have worked on.

Customer Churn Prediction using ANN

Technology Used: Artificial Neural Networks (ANN), TensorFlow/Keras, Scikit-learn, Streamlit

Description: This project predicts whether a customer is likely to stop using a service by analyzing behavioral and transactional patterns. Using an ANN model trained on historical customer data, it identifies key churn indicators, such as reduced engagement or irregular payments. It is used in telecom, banking, and e-commerce to predict customer churn and enable proactive retention with real-time insights via a Streamlit app.

SAR Image Colorization

Technology Used: GANs (Pix2Pix), TensorFlow/Keras, HTML, CSS, JavaScript, MongoDB, Flask

Description: Synthetic Aperture Radar (SAR) images are grayscale and difficult to interpret visually. This project develops a deep learning model using GANs to convert SAR images into realistic colorized optical images. By training on paired SAR-optical datasets, the model learns to reconstruct optical-like features while preserving structural details. This enhances the usability of SAR data in remote sensing applications like land cover classification, disaster monitoring, and urban planning.

Sentiment Analysis of IMDB Movies Dataset

Technology Used: NLP, LSTM, TensorFlow/Keras, NLTK, Scikit-learn, Streamlit

Description: This project analyzes the sentiment of movie reviews from the IMDB dataset, classifying them as positive or negative. Using an LSTM model, the system learns contextual dependencies in text, providing accurate sentiment predictions. It can be applied to movie reviews, customer feedback, and social media sentiment analysis. The project includes a Streamlit web interface for real-time review analysis.
